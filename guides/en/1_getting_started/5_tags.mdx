# Теги

Теги - это имена, метки или ключевые слова, которые группируют игровые объекты, такие как враги,
Друзья, деревья и т. Д. Они полезны для классификации различных типов объектов.

## Использование тегов

Теги могут быть добавлены в массив функции `add([])` как строка.

```js
const ghost = add([
    sprite("ghosty"),
    "enemy",
]);
```

Это делает Ghosty, имеющим тег врага, и есть много операций, которые вы можете сделать
с этим.

## Операции с тегами

### Как добавить тег

```js
// При создании объекта
const ghosty = add([
    sprite("ghosty"),
    "enemy";
]);

// Добавляйте 1 или много тегов!
ghosty.tag("friend"); // теперь друг :D
ghosty.tag(["friend", "invisible"]); // invisible forever
```

### Как удалить тег?

```js
ghosty.untag("friend"); // D:
```

### Как узнать есть ли какойто тег в этом game object ?

```js
ghosty.is("enemy"); // true если tag есть и false если нет
```

### Список всех тегов

```js
ghosty.tags; // массив всех тегов, только для чтения.
// result: ["enemy"]
```

<Info crew="gun" title="Important!">
Не пытайтесь push или modify этот массив. Он только для чтения. 
Если вы хотите добавить или удалить тег, используйте `.tag()` или `.untag()`.
</Info>

## Использование тегов в событиях и методах

Многие функции и методы поддерживают работу с тегами.

```js
obj.get("enemy"); // возвращает массив со всеми enemy game objects
```

## Особый тег: *

Все игровые объекты имеют тег `*` по умолчанию. Это может быть полезно, например, для
получения всех игровых объектов

```js
get("*"); // список всех game objects
```

## Компоненты как теги

В v3001, KAPLAY по умолчанию будет принимать component id как тег:

```js
// Это значит, что если у вашего объекта есть компонент sprite(), то также будет и тег "sprite"
const tga = add([
    sprite("tga"),
]);

tga.is("sprite"); // true
```

В следующих версиях это отключено по умолчанию, но если вы хотите включить это или
отключить в версии v3001, вы можете использовать опцию `tagsAsComponents`:

```js
kaplay({
    tagsAsComponents: false,
});

tga.is("sprite"); // false
```
